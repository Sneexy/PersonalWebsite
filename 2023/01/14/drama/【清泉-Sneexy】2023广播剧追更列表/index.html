# 【清泉 / Sneexy】2023 广播剧追更列表

备注：括号内为已听的期数

《海棠受的晋江就业指南》倒吊男X菜里有毒（第一期）
  • 【23.1.21】搞笑搞得有点平，买热搜那段太长了。先搁置着
《小后娘》敕勒川下X风允之（第一期10min）
  • 【23.1.25】《小妈》推荐的。发现是删减版。
《三妻四妾》正经太郎（役魏无双）X 矜持ING（役麒儿） & HolyNight &佐佐木&张小澈&小随（役赵唯一）&林予曦（役林齐）&小苒（第三期）
  • 【23.2.11】。后宫向，本来适合攻控，但是里面喜欢的受音又比较多。
  • 【第一期・沾女色】

看字幕认人，没认全。本期为青青草原攻吃醋，六人份的醋

  • 【第二期・美人局】，魏无双 X 司徒仕晨（CV 张小澈）

开始走剧情了，关系好复杂：司徒本来要和亲的，和亲是因为有人把他当他爹的替身；魏无双对麒儿形同父兄，麒儿对他爱得明显。然后，司徒和魏无双差不多一见钟情了，然后司徒为报恩情以身相许

  • 【第三期・你是我的唯一】，魏无双 X 赵唯一（CV 小随）。

感情线有点迷，听到一半还没咋的，受就非攻不嫁了，宁可做妾，这时候攻已经娶了几个了，还在追另一个人…… 觉得不能耽误受就没答应。最后是受被侵犯后攻提了亲，受于是觉得是为了安慰才娶的他

  • 【第四期・指腹为婚】，魏无双 X 林齐（CV 林予曦）

林齐是个身世凄惨的聋哑人，小哑巴嗯嗯啊啊的很可爱，心里说话配得也很乖。

哀其不幸怒其不争，他也太没脾气了，魏无双哄他做朋友就交了心，屡次找他都是为了让他救人，还学什么三从四德，医药事业也抛下了。师傅人挺好的，架不住徒儿胳膊肘往外拐 ==。

兔兔唱的 ed《玉良缘》好听，猫耳上是剧情歌，把歌切太碎了，纯歌版（音质差一点）

《08321》贾诩（役李嘉豪），林予曦（役赵明泽），刘明月（役黄博文） ，李兰陵（役周末/李兰陵），倒吊男（役张弛），冬冬（役陈逸）：★★★★（第六期，番外3）
  • 【23.2.1】 原创群像搞笑剧，为 CV 定制的，贾诩是江西方言，倒吊男是东北方言。冬冬久违的清冷受音，从声展就入坑这部剧了。
《闭嘴，你这假粉》龙盘 X 风允之：★★★★☆（第八期）

  • 主角：沈谓行 X 叶九月

  • 质量很高的搞笑剧，风允之又受又欠的样子。

  • 天空客串

  • 【第八期】

离上一期特别特别久。都快忘了剧情了。这一期攻的脑回路还是傻得很，两个人的感情线要收束了，可是用的是略微煽情的方式，所以虽然搞笑却没有前面那么有梗。整体还是搞笑的。

分手戏，理由和隋冬的如出一辙 hhh

这期攻的音量偏大。

《他们为什么仍在相爱》风允之（役肖照山）X 顾辰（役肖池甯）

  • 【23.2.8】

  • 【第一期】

肖池甯，疯批，狂躁暴力，习惯诱惑，阴鸷。外婆配音太年轻，情绪有些奇怪

  • 【第二期】

被攻包的实习生：“那个时候我就彻底懂了，对他这样的人动心，是会像你一样疯掉的。”
肖池甯：“是吗？…… 原来我疯了。”

都快忘了，胡颖雪是肖池甯的同学。

他妈看出来他的心思但没明说，并坦言他们俩夫妻就是合作关系，且至少暂时相爱，更狗血了，比起来还是他妈更有段位。（全员疯人）

肖池甯好诱受，发现激将法好用就又用了一次，虽然车非常短但两个人之间的拉扯有意思，最后一段步步紧逼，成功诱敌深入。

《无罪释放》杨东旭（役允落辰） X 陈想先生（役言欢）：（第二期）

  • 【23.2.9】 副 cp：风允之（役雷纪秋）X 卡修（役齐轩）

  • 【第一期】

言欢相对来说比较温柔，就算是放狠话也没有那种妖艳霸道的女王气质。

不明白为什么别人就不行，对允就可以直接交第一次。

也不明白为什么宁愿和猎鹰这样的狠角色接触也不想见到雷，只是愧疚的话不至于此吧。
（希望都是伏笔，别是 bug）

  • 【第二期】

时隔太久啥都忘了，开头是少年言欢和少年雷纪秋，另外的声优配的少年时期，不看字幕我还以为在讲别人的故事了。

故事本身比较复杂，人物也真的很多！猎鹰在跟踪出逃的攻受，为了让受帮他们做 假钞。现在受还喜欢 “哥哥” 雷纪秋，齐轩（CV 卡修）和受两个互相吃醋，攻表面为了帮受而去勾引齐轩。
笑意斑马貌似又是配的一对 cp？“斑：你不用跟我睡，我也会为你去死”，梦回《无耻之徒》
喜欢笑意在这部里的声音，又在撩人了（请在《无耻之徒》里复刻一下 hhh）

没有读原文也没有听姊妹篇，听起来有点难度，希望台词能口语化一点，还有译制腔和场景转换不清楚的问题，还好有配字幕，靠字幕看懂了一些情节。之后可能得先听姊妹篇或囤剧了。

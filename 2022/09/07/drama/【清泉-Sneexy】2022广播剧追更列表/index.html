# 【清泉 / Sneexy】2022 广播剧追剧列表

备注：括号内为已听的期数

# 追平

  • 《闭嘴，你这假粉》龙盘 X 风允之：★★★★☆（7）
    • 主角：沈谓行 X 叶九月
    • 质量很高的搞笑剧，风允之又受又欠的样子。
    • 天空客串
  • 《你怎么舍得让我的爱流向海》十六夜 / 擎苍 / HolyNightX 林予曦：★★★★（16）
    • 【22.2.27】主角：顾鹏飞 / 陈旭阳 X 苏锐
    • 狗血是狗血，但人物心理很真实，所以比 188 更气听众。
    • 苏锐的性格比较拽和暴躁，林予曦在这部里戏感挺好
  • 《像渡渡鸟一样死去》天空 X 倒霉死勒（1）
    • 【22.3.10】主角：陆博远 X 裴子安
    • 【第一期】没有预告那么虐，别是个小甜饼吧 ==
  • 《我是主角的恶毒继弟》Mr. 岑（1）
    • 【22.4.2】
    • Mr. 岑的受土萌土萌的；观望 ing
  • 《匿名投靠》倒吊男 X 矜持 ING（2）
    • 【22.4.20】主角：路时 X 徐寒珉
    • 故事偏沙雕向。两个人相处尬中有序
    • 只听过倒吊男的报幕，上一次听角色还是骚气的袁瑞，这次是内心沙雕咆哮帝，外人看起来可能是怪蜀黍
  • 《非正常契约夫夫》擎苍 X 心外無物（1）
    • 【22.6.1】主角：楚旌 X 暮轩然
    • 预告不错。剧情发展有点快，铺垫比较少。心外無物的声音虽然好听，但和预想的角色不太一样，比较软。
  • 《爸爸和爸爸》白玉京 X 林予曦（1）
    • 【22.6.23】主角：龙铁军 X 方原
    • 开幕雷击。两个主角 40 + 的恋爱戏，兔兔压低了声线还是很软糯。
  • 《我和替身渣攻恋爱后白月光回来了》续续点灯 X 陈小 C（1）
    • 【22.6.27】主角：江衍 X 程见渝
    • 双替身，白月光 CV 声声慢，这个配音配置就很吸引我。陈小 C 唯唯诺诺和霸气的样子都好听，喜欢听他的笑，好仙啊。
《替罪羊》轩辕靖日（役薛擎） X 陈想先生（役岑深）
  • 小 C 策划，又是一直虐，虐身虐心的剧。（攻方朋友惊叹收心梗虽迟但到）
  • 小 C 压低了嗓音收起了美人音，还有点弱受，意外地不排斥，在医院从抗拒到犹豫到接受的过程配得很到位。主要配角羊仔，HolyNight。

第一二期

  • 感觉配得最好的地方是，在医院岑深从抗拒到犹豫到接受的过程，配得很到位。

第三期上

  • 薛擎被拒绝了就跟顾姓替身好上了，然后说只是玩玩，回过头来又通过岑深的父母来接近岑深（还挺社牛的）。能找替身就说明不够爱，缺乏对感情的尊重，感觉两人关系的主动权一直在薛手上

第三期下

  • 这半期故事开始向苦情剧发展了😂，薛擎：就算影响你备考你也要给我做吃的，你就是爱我。
  • 在酒吧里薛擎也有点大少爷做派，自己去跳舞（说是给岑深看），把岑深跟狐朋狗友落一块儿，让他听见一些乱七八糟的过往。
  • 岑深一边儿觉得他不靠谱，一边被他洗脑自己是爱他的，结果他打电话说 “我爱你” 声音太大，被岑妈听见了，岑深还要反过来保证两人没什么。

  • 《他・他》倔强的小红，天空，郝凡（上期）

    • 【22.7.16】主角：舒朗，谈宁 / 谈臻，乔良宇
    • 原创剧，天空役双重人格，一个是女装大佬，一个是社恐程序员，设定比较有意思

  • 《无耻之徒》斑马 X 笑意（第一期）

    • 主角：姜晓宁 X 韩嘉；
    • 《法外之徒》的副 cp，看了小说很喜欢韩嘉，受了苦也不埋怨而是求出路。只更新到预告，小说鸽了十几年了，但是挺好看

  • 《落不下》赵成晨，倒霉死勒

    • 主角：段锐，段琰

  • 《宣扬中华文化》预告

  • 《满分赝品》第一期

    • 替身梗，陈小 C

  • 《08321》第五期 + 番外 3

    • 搞笑

  • 《溪亭日暮》第一期上

    • 陈小 C，古风甜饼
《没骨日》Holynight（役晁鸣） X 矜持ING（役姜亮点）：第五期

两个表里不一的疯子的爱恨情仇，有点青春疼痛文学的感觉，还狗血。矜持 ing 在剧里的大段大段的念白我很喜欢。

预告 & 第一期

  • 预告和第一期重合部分太多了，第一期前 4 分钟都是预告里的。中途 bgm 有点不搭。人设挺疯的。

第二期

  • 点点的大段大段的念白，青春疼痛文学，不过好听。Holynight 的声音有点亮亮的质感。

第三期

  • 回忆往事，夏天的西瓜水滩在地上，有点粘有点酸的感觉

第四期 & 第五期

  • 疯，两个人明里暗里的疯。破坏
《警官，借个胆爱你》唱油条（役罗战）X 天空（役程宇）听完第四期
  • 2022.10.28
  • 程宇大部分时候硬气，又有柔软脆弱的一面，偶尔会撒娇，天空的戏感很好～。罗战有点痞劲儿，和《年少与妖娆》、《想你的张青凯》里的老实样差别比较大，空条声音相性不错。笑意的声音也好好听。京腔。
  • 开头感情发展有点快，然后就是推拒追赶欲擒故纵然后在一起，吵架和好，有平淡的日常也有跌宕的情节。
  • 李宗盛和 SK 的几首 ed 好听

楔子

  • 京腔。空条耶。
  • 感情发展有点快

第一期

  • 天空撒娇～。程宇收买丈母娘。痞痞的油条，话唠厨师

第二期

  • 感情升温，矛盾化解。程宇是个哭包啊，说话语气把握地很好，但是战斗力还是阔以的。

第三期

  • 有声漫画耶。画风有点丑看着看着就习惯了。笑意的声音好好听

第四期

  • 笑意的声音真好听，花絮听他一直吐槽角色，可是这个角色被他配出来还是很好。天空的戏感真好，有一些脆弱又不失强硬。修罗场搞笑。
《人渣味儿》声声慢&林浪&蝎子莱莱&卡修×顾辰。听完第一期
  • 顾辰可爱的声线，恃宠而骄
《无罪释放》杨东旭 X 陈想先生：第一期

6park 链接，官方链接之后补

第一期

  • 言欢相对来说比较温柔，就算是放狠话也没有那种妖艳霸道的女王气质。
  • 不明白为什么别人就不行，对允就可以直接交第一次。
  • 也不明白为什么宁愿和猎鹰这样的狠角色接触也不想见到雷，只是愧疚的话不至于此吧。
    （希望都是伏笔，别是 bug）
《他们为什么仍在相爱》风允之，顾辰
  • （22.11.18）第一期：肖池甯，疯批，狂躁暴力，习惯诱惑，阴鸷。外婆配音太年轻，情绪有些奇怪
《猫腻》擎苍（役温海林） X 林放（役温瑞）：★★★★

link, 有早期日抓的感觉了，!! 高 h，车香，!! 还有感情线，两个人推拒的过程

追剧楼：

第一期

  • 肉比较香

  • 温瑞的猫腻

  • 温海林竟然在外面养跟他长得几乎一样的情人，还一年了，为什么还理直气壮的

第二期

  • 温瑞不是和顾瑞安做了交易吗？怎么还是他，不是他吧？但是他说 “不要抛弃我”，像顾，他内心炸毛，脾气不好，又像温。

  • 哦，原来前面还是他自己。

  • 要假装失忆吗

  • 哈哈哈，两边隐瞒，两边吃醋，妙啊。温瑞以为他没掉马，觉得他爸对顾瑞安好吃味儿。老男人的情趣。

  • 温瑞之前的演技，和温海林之后的演技，都挺好的啊 2333。

  • 小男人的情趣，故意让温海林吃醋

第三期上

  • 海报画得有点怪

  • 哦，原来 “不要丢下我”“不要走” 就是他自己的想法，温瑞一直害怕温海林不要他

  • ！！原来是因为温瑞怕痛，温海林不想做多了，才出去找别人，然后因为找别人，温瑞就很痛苦

《入赘》龙盘（役程翰良） X 斑马（役李琅玉）（听完两期）（催更！）
  • 感情线还没开始呢。斑马的声音偏平淡，有点翩翩公子的感觉，装出来的温和，但是又懂得世故，打架那里比较 A。虽然两个人的气场是攻要 “姜是老的辣” 一点，刺探人一套一套的，但每个回回合都是受得逞。不过第二期结束时，还是被攻摆了一道，循循善诱啊。
《每天起床都能看见男朋友在保养》赵成晨（役褚雁罗） X 风允之（役常曼）

1-5 期

  • 有些搞笑，不过也不太切题，就保养了几天罢了。
  • 第六期开始不夹了，开始平淡了。
  • 后期好棒 (＾◡＾) っ❤

7-8 期

  • 两个人都不太擅长吻戏和车呀
《百年不合》商桐（役霍学川） X 逆鳞无伤（役方知谨）：听完三期上
  • 【22.12.10】北南原著。副 cp：轩 ZONE（役谢经年） X 倒霉死勒（役元远）
  • 听的逆鳞的第二部剧，还以为他本来是攻音？原来这么萌，娇气得很，时而很欲。

# 广播剧单期剧评

注：非全一期的广播剧。剧评也可能包含几期一起。

注：可能涉及剧透

《营销号说你喜欢我 第一季》DK（役周秋隼）X 冬冬（役司镜若）

第一、二期

  • 冬冬的受音诶，这个声线我喜欢。有气质，不过内心 OS 是另一种风格，或委屈或搞笑或咆哮体，反差萌。冬冬的演技好好哦，很多情绪都拿捏得到位。
  • 剧的后期也不错。

第三期

  • 这一期周秋隼多少有点油腻了喔。司镜若，能屈能伸，就是脑子慢半拍。
  • 戏中戏里 DK 好 A，冬冬也是清冷路线，挺好的
  • 碎嘴八卦他们俩有一腿的工作人员，有点不适

第四期

  • 这一期比较平，周秋隼这莫名其妙地倒贴得有点严重。（不过弹幕剧透了原因
  • 直球克傲娇

第五期

  • 戏中戏冬冬配女儿，娇娇的。

  • 哦哟，冬冬的气息戏，司镜若这儿 OS 后说出来的话感觉欲拒还迎了。周秋隼真不把自己当外人！

第六期

  • 13'。喂饭那段比较搞笑，特别是内心 OS“撑死你撑死你……” 那句。
  • 揭开了一些剧院往事，司镜若和董晓伟的伤疤。（看弹幕才知道董晓伟的 CV 是郭浩然，又配了一个瘦音胖子哈哈，不怪司镜若说董看起来 120 斤，这听起来也只有 120 斤）
    不过他一直执着于即使牺牲重要的东西（比如感情）也要挽救剧院，另一边也这样要求司镜若，有点执念太深吧，毕竟人更重要。

第七、八期

  • 第 7 期过渡一下。

  • 第 8 期的戏中戏表现得太好了，一直到杀青出不了戏。冬冬演技真好，两个角色都很有代入感。

  • 后期优秀

第九、十期

  • 完结了，这两期司镜若老是装傻。收尾也有些仓促了，停在安戚的故事上。就从安戚这两期的戏份来看，也不完全是坏人，反而司镜若他爸清高得不切实际，行为又没有跟上。

各种花絮、戏中戏《烽火》未播片段

  • 这部剧的售后服务很饱满诶。戏中戏挺好的，中秋番外里 DK 的难过但不能崩溃的脆弱感不错，七夕的录屏和春节的沙雕，冬冬喊 “哥哥” 的合集太撒娇了

  • 导演用心了，花絮中的几个片段剧中确实能感受到，完成度很高。让小孩儿配音确实比很多夹着嗓子配的自然。

  • 再次表白冬冬的配音，受音太香了。其他 CV 也都挺好的

《无耻之徒》斑马（役姜晓宁）X 笑意（役韩嘉）

第一期

  • 千呼万唤始出来，从《不法之徒》就很喜欢韩嘉，《无耻之徒》原作 n 刷了。

  • 五年后的姜晓宁，看文的时候觉得是比较沉闷阴郁、玩世不恭、怀着执念的瘦子，感觉有点活泼了。不过毕竟是年下，一直比韩嘉天真。
    五年前的姜晓宁，又毛躁又脆弱，比较符合。

  • 韩嘉的声音好听，不过没有《不法之徒》那么惊艳，傲劲儿收敛了一些，有些温柔了。美人儿～

《大力出奇迹》夙七烈 X 冬冬

第一期

  • 比较搞笑，不按套路出牌。不过冬冬是攻音，战斗力爆表了。
《08321》贾诩、林予曦、倒吊男 X 冬冬、刘明月、李兰陵

声展 & 第一期

  • 不知道哪里 mark 来的，还是高优先级。应该是群像搞笑剧。
  • 冬冬的声展（役陈逸）循环了很多遍，这个声音，我的何故又回来了，比何故声线更高冷一点，很好听。贾诩（役李嘉豪）江西方言，听不懂，但搞笑。还是分不清李兰陵和锦鲤和斑马，刘明月也一股羊味儿，听少了

第 2-5 期 & 小剧场 1-2

  • 很难不怀疑是声优定制，人设分明，有梗，不过老是分不出黄博文（CV 刘明月），封面要是按出场的两两放一块儿可能容易识别一点。赵明泽（CV 林予曦）人设到底是花花公子还是纯情少年😂？
  • 小剧场里贾诩读诗很好听，气质（反差萌？

小剧场 3

  • 冬冬的青受音我好爱
《骑士的情书》擎苍 X 矜持ING

第九期

  • 袁瑞真是傻得可爱。委屈巴巴。郑秋阳也直得搞笑
  • 震惊，在这里也能听到小 C。柏图 29’（就两句话）
《爷们儿》Holynight（役严肆）X陈小C（役乔已）

第一期

  • 小 C 役舞蹈教练，气质拿捏得死死的，暧昧氛围也烘托得很到位。强强听起来就是舒爽。
  • 黑道老大暗恋卧底，还很不经撩，两次，没出息（bushi）

第二、三期

  • 整体后期还是挺好的，第一期的动次打次跟古典舞曲，后面的打斗场面烘托气氛，就是有时候声音太大了炸耳朵。
  • 乔已和妹妹都挺惨的，被自己的信仰背叛，确实很折磨，但即使这样还能为此背叛严肆，着实是舍己为人了。林纾也是身不由己，爱没有对错，她却爱上了深恶痛绝的人（不过收网时是不是不能直接把嫌疑人干掉，这儿是 bug 吗？）。现实真残忍。
  • 好在是 HE，“不要留光头了，哥喜欢长发美人。” 够霸气哈哈，感觉这个老大有点恋爱脑
《满分赝品》皮皮鲨 X 陈想先生（主角：章献怀X林冬迟）

第一期

  • Sandstorm 制作，连陈想的剧尺度都大起来了 [/ 允悲]，虽然只有事前事后，看见进度条上的唯一一张图片确认了没有完整版。
  • 披着狗血皮的纯爱故事，从车祸、失忆、替身开始，第一期就掉马了。节奏挺紧凑的，戏剧性也够，末了那句 “我的赝品，我得重新署上自己的名字” 也挺带感。
  • 林有点呆呆的但不笨，呆白甜，喜欢一个人的时候他赖床也可爱！跟人斗嘴也可爱！饿了偷吃玉米、被欺负了逃跑、时不时心虚、傲娇、被简单的示好感动，都挺可爱的～
《走狗》by商桐（役陆业征）X逆鳞无伤（役程展心）

第四期

  • 攻的声音不太男高
  • 误会还挺多，受为了不让攻受到牵扯所以也没太多解释，显得软弱 ​​​

第五期

  • 受爸也太渣了，对他儿子都这么狠。
  • 攻也忒有钱了，随随便便 30w 就拿来还赌债。这时候程展心就能皮了，还能管着他爸了。这俩高中生有些社会了，对钱也没概念似的。
  • YouTube 上的视频做得还挺用心的，还有片头、字幕。拉了兔兔来片尾报幕

第六期

  • 在一起之后，尺度突然大起来了。
  • 陆业征认真地叫他心心的时候，感觉很温暖，程展心终于被正常地爱着了。
  • 解气，程展心把他爹的骨灰扬水沟里了（这段的哭戏很真实
《艳鬼》by昱霖（役空华）&糖醋排骨（役桑陌）

第一期

  • 早期糖糖的高贵小美人音诶～
  • 剧情听得云里雾里的，看着弹幕 get 了一点点，还是很懵，大概是 “我爱你你却爱他，我为你付出了一切你却转手送给他” 的剧情

第二期

  • 一辆小车过去了。
  • 没有之前的记忆的空华，感觉是在抢回桑陌对则昀的爱，让他只看着自己。
  • 感觉角色太多了，很多都只出现了一下，也没有后文了（虽然可以衬托桑陌的好，以及空华对桑陌的弥补

第三期

  • 三百年前桑陌为则昀欠下的债，都在还着，最终由空华解了。不过桑陌还是不相信他了，单方面觉得他心里只有南风
《折枝/阶下囚》燃X陈小C （段凌X陆修文）

预告

  • 简直就是虐点集锦吧。

上期

  • 小 C 役的陆修文，（一如既往）是个嘴硬心软对重要的人特别有保护欲的傲娇，对弟弟和攻都特别好，还要不让对方太愧疚。
  • 可以理解为 “替身竟是白月光” 吗 [/ 笑 cry]

下期

  • 恃宠而骄的可爱劲儿，让人忍不住要宠他。然而只是爱而不得、苦中作乐。
    开个车被叫弟弟的名字；眼瞎了也要摸一摸复习，好在奈何桥不要忘记。末了说 “这一日怎么这样短”；让人心疼。

  • 很为弟弟和攻着想，让弟弟瞒着攻十年前的真相，让攻瞒着弟弟自己命不久矣免得让弟弟伤心，结果他一走，这两人都立马告诉了对方，真不让人省心。
    燃的哭戏很让人动容，更何况是这么好的修文。

  • 说好的这篇文一半虐一般糖，结果刚好停在了虐完的地方（宝宝心里苦

  • p.s 陆修文真的很喜欢笑，笑起来感觉世界都亮了

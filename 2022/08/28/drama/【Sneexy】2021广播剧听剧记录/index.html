# 【Sneexy】2021 广播剧听剧记录

# 年终小结：

  • 听中文广播剧的元年
  • 剧情第一，声音第二
  • 喜欢有真实感、细腻的剧
  • 一般是喜欢哪个声音就逮着薅，一路的墙头：
    • 姜广涛。实力派，但是后续没有发现风格对口的经典剧的主役受
    • 羊仔。挺可爱的，聒噪
    • 小随。戏感很好，受而不弱
    • 何故声线的冬冬
    • 陈张太康。挺甜的
    • 倒霉死勒。喜欢听他比较 man 声线的受。戏感挺好。听 cut 还挺可爱，声音和脸不贴
    • 天空。女王受，很喜欢天空配的江停

# 听剧列表

补记的所以内容比较少，想起了再补补吧

（🔖 豆瓣已标记）《作品名》；评分；主役；主角；简评（有条目的略）（n 刷 *）

  1. 🔖《将进酒 第一季》；★★★★★；袁铭喆 X 姜广涛；萧驰野 X 沈兰舟；关注姜 sir 的开始
  2. 《将进酒 第二季》；★★★★★；袁铭喆 X 姜广涛；萧驰野 X 沈兰舟；
  3. 🔖《迪奥先生 第一季》；★★★★；景向谁依 X 羊仔；张臣扉 X 焦栖；从羊仔最正经的剧入坑，关注羊仔的开始
  4. 《迪奥先生 第二季》；景向谁依 X 羊仔；张臣扉 X 焦栖；
5. 🔖《小白杨》；★★★★；八千里路X羊仔；俞风城X白新羽；
  • 广播剧好听，但是删减了很多细节，于是想象着羊仔的声音看了原作再回来听。广播剧给小说添彩了。
  • ed 也好听而且贴剧情。
  • 因为几乎没给俞第一视角所以忽略了很多他内心的纠结，本来是个立体的角色，一堆人在弹幕从头到尾刷 “渣” 很无聊。
6. 《一拜天地》；★★★☆；北落师门 X 小随；水三X秦书；
  • cv 声音很自然，而且受也不刻意文弱，喜欢这个声音。（关注小随的开始）
  • 情节有点少，没听够。
7. 《裸替》；★★★；图特哈蒙X羊仔；
  • 细讲应该有很多细节，但是每一个地方都给足了细节，反而导致情节太少，感情铺垫不充分。
  • 羊仔声音好听。
  1. [不更]《你却爱着一个傻逼》；夙七烈 X 卡修
  2. 🔖《一醉经年》；★★★★☆；轩 ZoneX 冬冬；宋居寒 X 何故；冬冬的声音太好听，喜欢何故，人间清醒，清醒着沉沦
  3. 《救赎》；★★★☆；胡良伟 X 小随；林奕 X 王邛；故事很短，但是感情很真切，小随声音好听，对 “那一刻，我真的好想活下去” 印象深刻
  4. 🔖《我嗑了对家 X 我的 CP 第一季》；★★★★；刘思岑 X 陈张太康；顾依凉 X 卫言梓；挺搞笑的，冬天听了心情会好。陈张太康的声音好甜啊。
  5. 《我嗑了对家 X 我的 CP 第二季》；★★★★；刘思岑 X 陈张太康；顾依凉 X 卫言梓；
13. 🔖《碎玉投珠 第一季》；★★★★☆；凌飞X陈张太康；丁汉白X纪慎语；
  • 真是温润如玉的一部作品，广播剧把剧情一点点铺开，就像是给你一点点看玉件的细节。
  • 配音和后期都很好，画面感很足，歌也好听。
  • 听花絮里边江导戏，说太康声音不要太虚，不虚才是理想中的状态，有些攻受分得太清都违和，除了可爱挂的，不太喜欢弱受。珍珠这个角色挺很立体。
14. 《碎玉投珠 第二季》；★★★★☆；凌飞X陈张太康；丁汉白X纪慎语；
  • 蜕变。
  • 整体还是比较顺利的，爱情事业两不误，很配。
  • 瞎眼张给老梁送行说等他自己百年之后没人配给他送行那段儿哭了。
15. 《活受罪》；★★★★；倔强的小红军X倒霉死勒；沈凉生X秦敬；
  • 虐点太高了没被虐到。不过配得还不错，没感觉原作很超神，
  • 音量有点没控制好，其他都不错。倒霉死勒喘得还不错～
16. 《王子病的春天 第一季》；★★★☆；倔强的小红军X倒霉死勒；谭睿康X赵遥远；
  • 喜欢这部里面倒霉死勒的声线。
  • 故事本身听到这还觉得不怎么虐，谁没有过失败的暗恋呢，就算是异性也不一定就能心想事成。
  • 细节蛮足的。
17. 《王子病的春天 第二季》；★★★★☆；倔强的小红军X倒霉死勒；谭睿康X赵遥远；
  • 日常太多，有点流水，不过中后期遥远哭的那几场挺好，角色还算丰满。
  • 梅梅声音好听，man，从前期到后期的变化也很到位
18. 🔖《网恋翻车指南》；★★★☆；袁铭喆X倒霉死勒；向淮之X景欢；
  • 反差萌。梅梅男声好听。
  • 感觉后期有点拉，场景切换有些生硬，有时候是语音还给打字的效果音。
19. 《将进酒 第三季》；★★★★★；袁铭喆X姜广涛；萧驰野X沈兰舟；
  • 原作、配音、后期、时长都挺好，制造了很多名场面。感觉姜 sir 有时候鼻音比较重。
  • 可惜了松玉。
  1. 《塑料姐妹花二三事》；★★★★；小随 X 倒霉死勒；；认真在搞笑，绝对是没听过的小随和倒霉死勒的风格
  2. 《金刚姐妹花二三事》；★★★；；；上一个的姊妹篇
  3. 🔖《彼得・潘与辛德瑞拉 第一季》；景向谁依 X 羊仔；谢竹星 X 王超；cut 是真逗，但正剧就有些无聊，搞笑是 cv 撑起来的
23. 《强制fq》；★★★☆；今宵X倒霉死勒；

故事虽小但比较完整，故事流畅，h 比较给力，就是受音太攻

  1. 《彼得・潘与辛德瑞拉 第二季》；景向谁依 X 羊仔；谢竹星 X 王超；成长了
  2. 《我的目标是不坐牢》；★★★☆；倒霉死勒 X 卡修；李璧 X 方沐；又爱撒娇又爱打直球的年下诱攻 VS OS 机受。还挺新奇的搭配
  3. 《招煮饭阿姨，男女不限》；★★★；小丑 X 小随；张来福 X 钟有礼；方言攻，感觉也不是很搞笑。小随声音好听
  4. 《话梅》；★★★☆；kinsenX 小 K；顾十一 X 顾丑；故事比较完整，小 K 的哭腔不错。这十一怎么一点信任都没有，好过分
28. 🔖《撒野 第一季》；★★★☆；倒霉死勒，涅槃；顾飞，蒋丞；

①对非原著党不友好，不好把握情节走向和主角性格，故事承接不够自然；（看评论说在猫耳删减过几次）
②既然后期蒋丞声音要软下去，一开始就不用沉那么低，最开头两个人声音有点像；梅梅声音好听；终于听到冬冬类似何故的声线，好听；
③顾飞还挺温柔，会织帽子、包扎，难以想象玩着消消乐、微信昵称小兔子乖乖还能当老大，怎么得到 “钢厂小霸王” 称号的

29. 🔖《撒野 第二季》；★★★★☆；倒霉死勒，涅槃；顾飞，蒋丞；
  • 渐入佳境，前面的铺垫用上了，感情线和故事线都展开了，人物性格也开始丰满了。
  • 编剧的删改有些挺一言难尽的
  • 导演和配音很优秀，倒霉死勒的声音本来就带着克制收敛，专注于自己的小世界，又乖又有耐心，有治愈的能力，很贴顾飞，顾飞本飞
30. 🔖《撒野 第三季》；★★★★☆；倒霉死勒，涅槃；顾飞，蒋丞；
  • 生日，吃醋，演奏，高考。
  • “后面有你，我就踏实；你睁开眼，我会听见。”
  • 蒋丞前进的时候，拉着顾飞也一起。
  • 剧情改得还是有点不连贯
31. 🔖《撒野 第四季》；★★★★★；倒霉死勒，涅槃；顾飞，蒋丞；
  • 失而复得。
  • 情绪捏得很稳，蒋丞的快乐到消沉，顾飞的消沉崩溃到自责，然后到最后的温暖。多亏蒋丞一路坚持，终于等到了顾飞睁眼
32. 🔖《破云 第一季》*；★★★★；阿春X天空；严峫X江停；
  • 天空声音好听。关注天空的开始；只在这部里 get 到景向谁依的戏感
  • 剧情还行，画个图才把关系理清了；
  1. 🔖《破云 第二季》；；阿春 X 天空；严峫 X 江停
  2. 🔖《破云 第三季》；；阿春 X 天空；严峫 X 江停

# 陈小 C 的声音总结

陈小 C 的声音有玉的质感，温润、干净、气质、高贵、自信的大美人，可以是女王受，虽然是纤细的受音，但可以很有气势。可以执着、腹黑、傲娇、冰冷，也可以天真、沙雕，风骚、闷骚。笑起来和撒娇的时候最让人受不住。

听过的剧里，他的演技都挺好的，虽然声线有限制，但是情绪很饱满多变，配得很自然。听过的剧里《困流》和《我只要你》角色性格比较立体，演技比较明显。

选剧也很对我胃口，大部分都挺虐的，就算是 he，也得虐过了再 he。而且还得是从过去，到故事开始，到故事结尾，一直虐。身心都要虐。典型的有《桃魅》。

很多人分不清他和林予曦，但是在我听来，一个是大家闺秀，一个是小家碧玉，陈小 C 虽然是受音，但攻气十足，高傲自信；林予曦的声音就没什么攻击力，软软糯糯像个小白兔，让人想 rua，发脾气跟撒娇卖萌似的，很多时候角色自卑内省。

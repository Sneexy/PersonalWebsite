  • 好听剧，不求甚解，每有会意，便欣然忘眠
  • 受控，喜欢女王受、美人受、软糯受、萌受……

# 墙头们

本命天空 & 陈小 C，墙头（专注作品，时间顺序）：

  • 旧旧墙头：神谷浩史，平川大辅，下野纮，野岛健儿，武内健

  • 旧墙头：羊仔、倒霉死勒

  • 今年的新墙头 ：

    • 林予曦。软糯又带点气质，很适合配有个性的角色，倨傲或乖顺忍耐的都好。选剧本不太行。唱歌好听。没有偶像包袱，剧里该破音就破音 hhh

    • 糖醋排骨。戏路广，演技很好，很贴角色，能配出角色的复杂性。戏外又很萌 。唱歌好听

    • 矜持 ING。声音有种魅惑的美。笑起来很好听。eng 总读成 en。

    • 陈小 C。现在叫陈想先生。声音的底色像玉一样，演技很好，很多情绪个性都能拿捏，淡淡的但也挺自然。配虐剧很有感觉，傲娇闷骚矜贵冰冷腹黑美强惨专业户。比林予曦攻多了。po 总读成 pe，bo 总读成 be。（升级了）

《双生梅》天空X倔强的小红：★★★☆（1）

《坏故事之监禁》心外無物，雅策君：★★★★（1）
  • 【22.5.13】主角：林志，叶杰
  • 单箭头的爱，疯狂的迷恋，爆发的占有欲，编织的谎言，从天堂到地狱。叶摆脱的是束缚，也是无条件的、事无巨细的 “我好爱你”，从他闪耀的少年时代，到他发福谢顶被生活摧残到没有激情。
  • 心外無物配得很好，自己的爱被看到时的激动兴奋，语速也变快，能听出他眼里的光；非常小心翼翼，卑微，只要叶愿意他能付出一切。因为占有欲而疯狂，语无伦次，蹦出来的话沾上了很浓烈的感情。
《社交温度》倔强的小红军X林予曦：★★★☆（6）
  • 【22.5.15】主角：宋远旬 X 方昭暮
  • 兔兔声音很软，可爱，他跟 Andrew 说话真的好甜。 “你的喜欢值钱，我的喜欢不值，我给 Andrew 的喜欢呢，我去找谁要回来啊？” 听到这里的哭腔就能体会方的伤心了，心疼。
  • 不太喜欢的点是，最开始宋误会方抢了他朋友的交换名额、以及觉得他娘而带头孤立他，又因为喜欢他才在现实中缓和关系。另外，方在国内有那么多朋友，不至于只能找 Andrew 倾诉吧。
  • 3 话弃了，猫耳上架又捞起来下饭了，听了宋的解释好像又能接受了。故事整体还是有点别扭。
[不更]《关河令》陈小CX矜持ING：★★★☆（1）
  • 【22.5.16】主角：毓弋 X 怜更
  • 非常古早，听得不清晰。不过故事和矛盾已经铺垫出了，画面感也比较足。感觉是虐剧。
  • 陈小 C 攻，锦鲤攻
  • 番外《客官不可以》矜持 ING 饰 250 小倌，陈小 C 饰妖娆小倌，向以辰饰倒霉嫖客。听了很多遍。六分钟的小短番外。陈小 C 和矜持 ING 是够妖娆和沙雕的，听矜持叫妈妈。

《疯子》Mr.岑X心外無物：★★★★（1）
  • 【22.5.21】主角：兔 X 叶梓
  • 超长全一期。剧如其名，越听到后面越疯越带感越压抑。从兔读诗的那段开始，故事推向高潮，听完有点难受。（这首诗感觉以前在哪里听过，唱的或者很 drama 的形式念的）
  • 心外無物的戏感很好，Mr. 岑配病娇的时候比一般的男声时好一些。
《年少与妖娆》唱油条X陈小C：★★★★☆（5）
  • 【22.5.23】主角：易坤 X 卓影
  • 剧情、人设、后期都挺带感。黑道背景，女王受 & 忠犬攻。反转很多。结局还挺欣慰的，他们都守住了自己的承诺。易坤看着呆呆的，最后其实是在炫耀他赢了吧，所以卓影才嘴硬。
  • 《年少与妖娆》指《Young and Beautiful》，其实为了《破云》的 “仲夏初茫，七月未央” 就听过很多遍这首歌，但是他们之间没有双向的感情，只有执念和憎恨。而这部剧里这首歌作为插曲和 ed，更为贴切，虽然 be 了，他们自始至终是愿意为对方付出生命的，比黛西和盖茨比的感情更动人，每一期分隔得刚刚好，ed 所承载的情绪不一样。
  • 整部剧主协役龙套的戏感大部分都挺好，声音也好听（俩主役的声音让人心动，听习惯了才被拉回剧情（可能也有张青凯的滤镜在））。卓影的妖媚、女王、A、深情，易坤的忠诚、木讷、木讷地撒娇都挺好，而且不同心境下的变化也表现得很好。还有轩辕靖日、魅影之声、HolyNight、十二月等，续续点灯报幕。

《当我用灵魂和恶魔做了交易之后》擎苍，星潮：★★★☆（1）

《死对头是猫薄荷而我是猫肿么破》轩ZONEX萧翟：★★★★（16）
  • 【22.6.25】主角：阚泽 X 司景

  • 【22.6.29】主角：秦浪 X 林羡

《Shawn》商桐X林予曦：★★★★（4）
  • 【22.6.29】主角：“我” X Shawn
  • 因为女装和林予曦，被朋友反复安利反复心动 mark。故事的开头挺惊艳的，Shawn 的高跟鞋、旗袍、林予曦的不完全的伪音、叫 “Daddy” 和 “先生” 的声音戳到我心里了。唱的《秦淮景》很惊艳。
  • 故事有些唏嘘，最开始他可怜 Shawn 和母亲一样的遭遇，怜悯而后爱上 Shawn，Shawn 陪他度过低迷，为他争取机会，他确实让两人过上了好的生活，可 Shawn 的重点是 “和他一起” 过这样的生活。Shawn 挺重情义，被他抛弃就听他安排，意外是为了找他，遗产受益者也是他。而他只在失去时后悔，在事业受挫时才为此伤心（和律师交接是找助手处理的），打着殉情的幌子掩盖他这一生方方面面的失败。前妻 Jennifer 除了眼光不行别的也挺好（CV 黑芝烧声音好听）。
  • 剧给文锦上添花了，第四期后期很棒。剧末的彩蛋是广播剧福利，把 be 变成 oe 了。但听完还是有些难受。

  • 配音：小 C 的声音和选剧都挺合我的胃口的，这部感觉氛围和《活受罪》有点像，但是比起梅梅，会更容易心疼小 C 配出来的角色（是滤镜吧），求而不得，哀而不伤，（外加一点诱），反而是末了燃的哭戏情绪比较浓重一些。斑马配的柳逸也蛮可爱的，虽然菜但讲义气
  • 看上去倨傲而恃宠而骄，其实是口是心非，非常善良、舍己为人的角色。为他折一枝桃枝，却像是给了他来年的春天。

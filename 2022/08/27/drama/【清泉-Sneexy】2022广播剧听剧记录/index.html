  • 好听剧，不求甚解，每有会意，便欣然忘眠

# 墙头们

本命天空 & 陈小 C，墙头（专注作品，时间顺序）：

  • 旧旧墙头：神谷浩史，平川大辅，下野纮，野岛健儿，武内健
  • 旧墙头：羊仔、倒霉死勒
  • 今年的新墙头 ：
    • 林予曦。软糯又带点气质，很适合配有个性的角色，倨傲或乖顺忍耐的都好。选剧本不太行。唱歌好听。没有偶像包袱，剧里该破音就破音 hhh
    • 糖醋排骨。戏路广，演技很好，很贴角色，能配出角色的复杂性。戏外又很萌 。唱歌好听
    • 矜持 ING。声音有种魅惑的美。笑起来很好听。eng 总读成 en。
    • 陈小 C。现在叫陈想先生。声音的底色像玉一样，演技很好，很多情绪个性都能拿捏，淡淡的但也挺自然。配虐剧很有感觉，傲娇闷骚矜贵冰冷腹黑美强惨专业户。比林予曦攻多了。po 总读成 pe，bo 总读成 be。（升级了）

  • 总结：13、14 年的剧，剧情台词设计得都挺流畅的，后期挺好的，氛围到位。
  • 剧情：应该是年下，小园有些幼稚但有年轻人的朝气，也很坦诚，（还是个嘤嘤怪老是向老婆撒娇）；海唐被家庭抛弃所以敏感自卑，发现感情淡了就开始别扭了，产生了疏离感。好在一个抵挡得住诱惑（内心 OS：我家海唐比你高比你壮比你好看比你体贴，滤镜开满），一个愿意信任。吴筱山（陈小 C 役）主动、绿茶式温柔、貌美、大长腿，但是强迫不可，还自欺欺人，最后被小园拿开水瓶打了，还被海唐揍了，海唐揍人的时候还挺帅气，一揍把小园的不安给揍散了（CC 的早期反派美人炮灰攻，末了还说：祝你俩白头偕老断子绝孙）。
  • 角色 & 配音：整体演技挺好，入戏。刚开始有点分不清冬冬和乘风，后面就靠性格区分，一个呆愣直率，一个成熟稳重但内心自卑又不愿意表达（评论区说秦川，现在靠吐词听得出来了，之前想成朴枫了我的锅）。小 C 配的吴筱山正常的样子还像个人，追求的时候的自信和撩拨，强迫的时候的极度自我，事后对着小园装无辜，对着海唐绿茶，配得好。风雨归舟配的常辞傻兮兮的；陆念 CV Mic 配的弱智小孩，不过挺可爱的；木野信配的弟弟孟小衡，刚开始看着不闻不问的，遇到事很有兄长风范，可靠。夫夫相性问答整得跟真的访谈一样，有种他们真实存在的感觉，小园的 n 次夫唱夫随 “就是，么么哒～”。
  • 海唐很温柔，给陆念编织了一个梦，梦里他就是自闭症大雄。有海唐这个机器猫，陆念一定能成为一个勇敢的孩子。
《他的礼物》郝凡 X 小狼：★★★（全一期）
  • 【22.9.20】主角：季钦 X 泽安
  • 那两年的泽安其实挺好的，替他铺好一切，最后还自责抑郁了很长时间。原本的泽安有些不值，家人又不是不能接受他，朋友不习惯也可以远离，季钦不爱他就分吧，太跟自己较劲了，一定要获得别人的认同，可惜，毁了自己也毁了替身的用心良苦。
  • 主役声音好听
《营销号说你喜欢我 第一季》DK（役周秋隼）X 冬冬（役司镜若）：★★★★（全十期）

  • 整体配音、导演、后期都挺到位的。很多细节很自然又很让人入戏，人物也比较饱满有特点。不过最后几期收尾略仓促，故事还没讲完，连标题都没切到，配角戏份比较重。

  • 最喜欢的是冬冬的声音和戏中戏《烽火》。

    • 很喜欢冬冬的受音，这个角色还比较立体，不是傻白甜，会撒撒娇卖卖萌，但是脑子还比较清醒，看着淡定但内心 OS 有很多委屈 / 搞笑 / 咆哮体之类的。整体演技都挺好，几个小孩儿还真找了小孩来配，戏中戏里冬冬和 DK 的演技忒好了。

    • 《烽火》背景是民国时期，逻辑比较顺，be，养兄弟之间的感情很真切，比剧本身更有实感。

《犹记斐然》弦小雾（役游信） X 朴枫（役季斐然）：★★★（全一期）
  • 误会。弦小雾声音好听，雅致，朴枫也不错，不过还是没听出来。
  • 在猫耳没字幕地听完了，剧情没完全 get 到，结果发现 B 站有字幕且弹幕多很多。看了才明白误会在哪里，可是两个人相爱，总不至于一丝痕迹也没有的。
《阶下臣》by风允之（役赵驰）&林予曦（役何安）：★★★★（全十六期）
  • 制作精良，后期、配音、配乐、音效之类都挺好的。就是太长了，十六期，一期还分上下，听到后面又疲惫了。一直在搞事业，前期感情线几乎一直单箭头，等报完仇就为了感情把事业全放弃了，有点草草收场的感觉。
  • 何安一直过于卑微，对心上人很执着，对手下仁慈，但也做了很多残忍的事。
  • 有天空～～。彭尧配的王阿声音好听，意难平，从头到尾都是为了何安，何安还知道他爱而不得。

单期剧评

第八期

  • 这期基本能盲听。
  • 一心搞事业，第八期了，还在一心搞事业，感情线终于有一点眉目了。事业线的逻辑还是比较通顺的。
  • 天空还是演技在线，娇媚和正气平衡得刚刚好。
  • 王阿有点意难平啊，一直对何安这么好，何安却没会意
  • 弹幕把 “厂工” 叫 “厂花” 可还行

第九期

  • 高兴到破音，熟悉的破音哈哈。
  • 采青一事，不论是不是注定，只要何安为了达到目的递了刀子，都有错。他对盈香如何，采青也是别人的意中人、别人的女儿姐妹啊。所以何安说自己 “虚伪至极” 也没错

第十期

  • 4' 处，天空的报幕好媚啊。主 cp 的感情线已经够慢的了，副 cp 也这么慢。
  • 车开起来了。不过按这个节奏看上去是应该 be 啊，搞了这么久事业，好歹甜一会儿吧

第十一期

  • 有一点听疲了，听古耽最不适应的就是尊卑，何安太卑微了。不过哭哭啼啼的怪可爱的。
  • 适时地开了一点车。

第十二期

  • 协役群杂音效后期依然在线。
  • 上期末尾何安和时开的对话，还挺稳的，有些 A 了，不过时开一走就泄气了，挺不容易的。
  • 本期搞事业 + 回忆。感情线着实比较少耶

第十三期

  • 上下两期分三次听完的，前面有点儿记不清了。何安受了好些苦，然后守得云开见月明，至少两情相悦了
  • 王阿坐视不管，要敲打何安，也没有分寸了。不过何安倒也一直没有把他当过自己人

第十四期

  • 前面两人密谈篡位声音未免太大了 [汗]，何安都喊出来了。
  • 这回王阿把当年的实情都说出来了。挺唏嘘的吧，他为了何安做这些得罪了赵驰，最后何安看也不看他一眼，把他的好当做理所当然，掏心掏肺只给赵驰，着实不甘心。到最后王阿却还在帮他

第十五期

  • 感觉篡位篡得有点容易了，一个二把手太监可以明目张胆弑君？流放的皇子哪来的那么大的兵权？皇帝、太子和万贵妃都丝毫不懂权谋的样子。

  • 算是两个人恋爱脑孤注一掷了，要不是双向奔赴手下都得 gg。反正不论过程，结尾是好的

第十六七

  • 完结撒花。给所有人物都安排了结局（不过都集中定下来，有点儿为结局而结局的感觉）。平安喜乐。
85.[不更]《大神养成计划》阑珊梦（役卫鸿） X 笑意（役段寒之）：★★★☆（五期）

古早剧，淮上原著，剧情流，娱乐圈。比起主角之间的感情，刻画更多更深的是段寒之和关靖卓的感情，因为误会而分开。段寒之霸气又霸道，如果不是笑意这样的女王美人音，可能实在接受不了。推动剧情的生病受伤莫名其妙就好了，娱乐圈的部分比较胡闹，不过段寒之导演时的凶劲儿倒是不错。

第一期

  • 淮上原著耶。段寒之挺 A 的，霸气，不好不坏

第二、三期

  • 旧情，越看越觉得攻没什么存在感，反而是关靖卓戏份比较重，感情比较深

第四期

  • 走剧情走得有点离谱了，得病受伤总是没事儿，段寒之和卫鸿之间还不如和关靖卓的感情戏深。
